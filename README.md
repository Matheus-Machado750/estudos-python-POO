# Estudos Python - Programação Orientada a Objetos (POO)

Este repositório reúne **exercícios, experimentos e mini-projetos desenvolvidos em Python com foco em Programação Orientada a Objetos (POO)**.  
O conteúdo foi construído de forma progressiva, com o objetivo de consolidar os principais conceitos da orientação a objetos e aplicá-los em cenários práticos.

O repositório faz parte da minha evolução técnica em Python, representando uma etapa posterior aos estudos de lógica e fundamentos da linguagem.

---

## 🎯 Objetivos do Repositório

- Consolidar os **fundamentos da Programação Orientada a Objetos em Python**
- Praticar a modelagem de problemas utilizando classes e objetos
- Aplicar conceitos como encapsulamento, herança, polimorfismo e composição
- Desenvolver código mais **organizado, reutilizável e escalável**
- Servir como material de estudo contínuo e **parte do meu portfólio público**

---

## 🧠 Conceitos Trabalhados

Ao longo dos exercícios e exemplos, são explorados temas como:

- Definição de classes e objetos
- Construtores (`__init__`)
- Atributos e métodos
- Encapsulamento
- Herança
- Polimorfismo
- Composição
- Organização de código orientado a objetos
- Boas práticas em Python

---

## 📂 Estrutura do Repositório

No momento, os exercícios e exemplos são adicionados de forma incremental, **sem uma estrutura rígida inicial**, priorizando a prática e a consolidação dos conceitos de Programação Orientada a Objetos.

À medida que o repositório evolui e a quantidade de exercícios aumenta, os arquivos passam a ser **organizados por temas e níveis de complexidade**, refletindo a progressão natural do aprendizado.
